# VISION — SkillHub

Internal AI skills marketplace enabling org-wide Claude skill sharing.

## North Star

Every employee discovers, installs, and contributes Claude skills as easily as installing an app — with division-level access control, quality gates, and community engagement.

## Core Principles

1. **Division-first security** — access control at every layer, never client-side
2. **Quality through gates** — schema validation, LLM judge, human review
3. **Developer-native install** — MCP + CLI, not just web UI
4. **Community-driven curation** — reviews, ratings, trending, featured
5. **Spec-driven development** — design docs before code, always

## Success Metrics

- Skill discovery time < 30 seconds
- Install via CLI in 1 command
- 80%+ test coverage enforced
- Zero unauthorized cross-division access
- Submission-to-publish < 48 hours (with gates)
