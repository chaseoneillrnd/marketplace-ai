# agentic:repo:quality:map

No questions. No confirmation. Run immediately.

## Phase 1: Discover Structure

```bash
# Apps / services / packages
ls -d apps/*/ 2>/dev/null
ls -d services/*/ 2>/dev/null
ls -d libs/*/ 2>/dev/null
ls -d packages/*/ 2>/dev/null
ls -d src/*/ 2>/dev/null

# Manifests for each discovered directory
find apps libs packages services src -maxdepth 2 \
  \( -name "package.json" -o -name "pyproject.toml" \
     -o -name "go.mod" -o -name "Cargo.toml" \) \
  ! -path "*/node_modules/*" 2>/dev/null
```

**For each app / lib / package discovered:**
1. Read manifest file for name and description
2. Detect framework from dependencies
3. Detect port (scripts, env files, config)
4. Classify as `app` (has entry point/server) vs `lib` (imported by others)

**Detect dependency relationships:**
```bash
# TypeScript/JS: tsconfig paths or package.json workspaces
cat tsconfig.base.json 2>/dev/null | grep -A 20 '"paths"'
grep -r "workspace:\*" apps/*/package.json libs/*/package.json 2>/dev/null

# Python: requirements
find . -name "requirements*.txt" ! -path "*/node_modules/*" 2>/dev/null | head -3 | xargs head -5

# Go: go.mod require
find . -name "go.mod" ! -path "*/vendor/*" 2>/dev/null | xargs grep "require" -A 10
```

**Detect key file locations:**
```bash
# Entities / models
find . \( -name "*.entity.*" -o -name "*.model.*" \) \
  ! -path "*/node_modules/*" 2>/dev/null | head -5
find . -type d \( -name "entities" -o -name "models" \) \
  ! -path "*/node_modules/*" 2>/dev/null | head -5

# Migrations
find . -type d -name "migrations" ! -path "*/node_modules/*" 2>/dev/null | head -3

# Config
find . -name "*.config.*" -maxdepth 3 ! -path "*/node_modules/*" 2>/dev/null | head -8

# Environment templates
find . \( -name "*.env.example" -o -name "*.env.template" \) 2>/dev/null | head -3
```

## Rules

- Never ask questions. Read → infer → write → report.
- Use actual detected values. Omit columns/rows when data is not available.
- Create `docs/agentic/` if missing — do not fail because it doesn't exist.
- Keep the map navigational, not exhaustive. One row per top-level entity.
- For monoliths with no lib system, replace the Libraries section with a `src/` subdirectory table.
